Frequently Asked Questions

Why do you ask for an API key?

The GG.deals API key is completely optional. Adding your own key helps reduce rate limits if they become an issue and provides more reliable access to pricing data. Without a personal key, LootScout uses a shared proxy server.

A game is not showing updated prices. Why?

LootScout relies on GG.deals for pricing data, and sometimes deals are not instantly updated when sales go live. Check back in a few minutes or hours for the latest pricing information.

How does the rarity system work?

The 8-tier rarity system rates deals based on discount percentage, from Common (0-14% off) to Iridescent (100% free). Optional modifiers can adjust ratings based on review scores and average playtime.

What data does LootScout collect?

The only data sent out is the Steam game ID to retrieve pricing information. No personal information is collected or transmitted to external servers. All processing and analysis happens locally in your browser.

How often is the data updated?

LootScout uses a 30-minute cache system for optimal performance. When you visit a game page, if cached data exists and is less than 30 minutes old, it will be displayed. Otherwise, fresh data is fetched from the APIs.

Why is my language/region not available?

Available regions and languages are limited by the API that LootScout relies upon. We support all regions and currencies provided by the GG.deals API. If your region isn't listed, it may not be supported by the underlying data provider.

Why don't some games show playtime data?

Playtime data is calculated from Steam user reviews, but some games don't have sufficient review data to calculate average playtime. This commonly affects free-to-play games and DLC content, which often have limited or no playtime information available from Steam reviews.
